Home security system

This is a raspberry pi project to prevent burglary - Home security system
The 3 main sensors of this system are 1) IR sensor 2) DHT22 3) MQ-6 . IR sensor detects if there is any movement . DHT22 helps monitor the temperature and humidity of the system. and MQ-6 helps detect if there is any gas leak in our house This whole data is logged into the website created with help of flask and data keeps on updating itself , so we can keep a check .

Lithium Ion battery charger

This circuit helps recharge lithium ion batteries.
The main component of this circuit is LM317. The LM317 is an adjustable three-terminal positive-voltage regulator capable of supplying more than 1.5 A over an output-voltage range of 1.25 V to 32 V. Also darlington set of transistors is used .A Darlington pair behaves like a single transistor in such a way that the current amplified by the first transistor is amplified further by the second one. which helps charge out Li-Ion battery
